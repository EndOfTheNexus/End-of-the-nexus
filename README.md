# End of the Nexus

End of the Nexus is a dark fantasy action RPG made by Jonah Matthew Moore.

It includes:
- Story Campaign
- Boss Rush
- Zombie Apocalypse
- Survival Mode
- Training
- Multiplayer
- Premium Ticket features

## Main Files

- `index.html` - main game page
- `script.js` - game code
- `style.css` - game styles
- `how-to-play.html` - how to play page
- `privacy.html` - privacy page
- `updates.html` - news and updates page

## Run Locally

Open `index.html` in a browser.

For the best app/install behavior, run it from a local server or a real website host.
